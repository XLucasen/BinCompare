# 快速参考卡片

## 文件加载方式

### 方式1：按钮选择（传统方式）
```
点击"选择文件A" → 选择文件 → 打开
点击"选择文件B" → 选择文件 → 打开
```

### 方式2：拖拽加载（新功能）✨
```
拖拽文件 → 放到左侧区域（文件A）
拖拽文件 → 放到右侧区域（文件B）
```

## 键盘快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+O` | 打开文件A | 打开文件对话框选择文件A |
| `Ctrl+Shift+O` | 打开文件B | 打开文件对话框选择文件B |
| `Ctrl+H` | 切换显示模式 | 十六进制 ↔ 二进制 |
| `Ctrl+E` | 导出差异 | 导出差异报告到文本文件 |
| `Delete` | 清除所有 | 清除已加载的文件和对比结果 |

## 显示模式

### 十六进制模式
- 格式：`48 65 6C 6C 6F 20 57 6F 72 6C 64 21`
- 优点：紧凑、易读
- 用途：一般对比

### 二进制模式
- 格式：`01001000 01100101 01101100...`
- 优点：显示位级细节
- 用途：位级分析

## 每行字节数

| 选项 | 用途 | 场景 |
|------|------|------|
| 8字节 | 详细查看 | 精确定位小差异 |
| 16字节 | 平衡模式 | 默认选项，最常用 |
| 32字节 | 快速浏览 | 一次看更多数据 |
| 64字节 | 大文件 | 快速定位大范围差异 |

## 拖拽加载视觉反馈

### 拖拽进入
```
文件A区域 → 边框变蓝色 + 加粗
文件B区域 → 边框变绿色 + 加粗
```

### 拖拽离开
```
边框恢复灰色
```

### 放下成功
```
状态栏显示："文件X已加载: filename.bin (size 字节)"
自动对比（如果两个文件都已加载）
```

## 常见操作流程

### 快速对比两个文件
```
1. 拖拽文件A到左侧区域
2. 拖拽文件B到右侧区域
3. 自动进行对比
4. 查看差异结果
```

### 切换显示格式
```
按 Ctrl+H 或 点击"切换为..."按钮
```

### 查找特定差异
```
1. 在右侧差异列表中找到差异
2. 点击地址（蓝色下划线）
3. 自动跳转到该位置
```

### 导出差异报告
```
按 Ctrl+E 或 点击"导出差异"按钮
选择保存位置
生成差异报告文件
```

### 重新对比
```
1. 点击"清除"按钮或按 Delete
2. 加载新文件
3. 自动对比
```

## 差异类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| 数据不同 | 两文件在该位置字节值不同 | A: 0x48 vs B: 0x49 |
| 文件A超出 | 文件A比文件B长 | 地址超过B的长度 |
| 文件B超出 | 文件B比文件A长 | 地址超过A的长度 |
| 超出范围 | 两文件都结束但长度不同 | 末尾标记 |

## 常见问题快速解答

### Q: 拖拽不工作？
A: 确保拖拽的是文件，不是文件夹

### Q: 能否拖拽多个文件？
A: 不能，只加载第一个文件

### Q: 拖拽后为什么没有自动对比？
A: 需要两个文件都加载后才会自动对比

### Q: 如何快速切换显示模式？
A: 按 `Ctrl+H` 快捷键

### Q: 如何导出差异？
A: 按 `Ctrl+E` 或点击"导出差异"按钮

### Q: 如何清除所有数据？
A: 按 `Delete` 或点击"清除"按钮

## 推荐工作流

### 场景1：快速对比两个文件
```
1. 打开文件管理器
2. 拖拽文件A到左侧区域
3. 拖拽文件B到右侧区域
4. 查看差异结果
5. 点击差异地址快速定位
```

### 场景2：详细分析差异
```
1. 加载两个文件
2. 按 Ctrl+H 切换到二进制模式
3. 调整每行字节数为8
4. 点击差异地址逐个查看
5. 按 Ctrl+E 导出报告
```

### 场景3：批量检查多个文件对
```
1. 对比第一对文件
2. 按 Delete 清除
3. 拖拽第二对文件
4. 重复步骤2-3
```
